Project information

  • Project Name: PUZZLE-DICT T | Application | Individual |
  • :Windows Form, C#, ADO.NET, SQL Server
  • Project URL: www.example.com

Key responsibilities:

  • Puzzle-Dict application is having two functionalities. In this application I made two special options one is for English word dictionary and second is for word puzzle.

  • In this project I have made word shuffle game in window form and for playing this game particular time slot will be provided to the user

  • Words will be loaded from an external text file, We will add those word to the text file using C#.

  • Each word in the text file will be saved as a list item then we will create a scramble function that will shuffle or scramble the words around in a random order and display it on the screen

  • There is one textbox in the application for accepting input from user and after that it will captured the input and compare it with the original word.

  • If the captured word is correct then it will move towards next word and shows 1 point in correct label’s textbox otherwise shows zero in incorrect label’s textbox, and this will be increases and decreases according to the answer.
